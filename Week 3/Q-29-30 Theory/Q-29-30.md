# Q29. Sensor Calibration

## What is Sensor Calibration?

**Sensor calibration** is the process of comparing a sensor's output with known reference values and adjusting it to improve measurement accuracy. Calibration reduces errors caused by manufacturing variations, environmental conditions, or sensor aging.

### Why is it Important in IoT?

- Improves measurement accuracy.
- Ensures reliable sensor readings.
- Reduces measurement errors and drift.
- Increases the reliability of IoT monitoring and automation systems.

## Calibrating an Analog Soil Moisture Sensor

A common method is **two-point calibration**:

1. **Dry Calibration**
   - Keep the sensor completely dry (in air).
   - Record the analog value as the **dry reference** (e.g., 1023).

2. **Wet Calibration**
   - Place the sensor in fully saturated soil or water.
   - Record the analog value as the **wet reference** (e.g., 300).

These two values are then used to map future sensor readings to a **0–100% moisture level** using the `map()` function or a linear interpolation formula.

# Q30. I2C Protocol

## What is I2C?

**I2C (Inter-Integrated Circuit)** is a synchronous serial communication protocol that allows multiple devices to communicate using only **two wires**.

## SDA and SCL Lines

- **SDA (Serial Data Line):** Transfers data between the master and slave devices.
- **SCL (Serial Clock Line):** Carries the clock signal generated by the master to synchronize communication.

## How Does I2C Addressing Work?

- Every I2C device has a unique **7-bit or 10-bit address**.
- The master sends the address of the target device.
- Only the device with the matching address responds, while all others ignore the transmission.

## Common IoT Sensors Using I2C

1. **BMP280** – Temperature and Pressure Sensor
2. **MPU6050** – Accelerometer and Gyroscope
3. **BME280** – Temperature, Humidity, and Pressure Sensor
