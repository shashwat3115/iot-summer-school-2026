# Q12. Microcontroller vs Microprocessor

The following table compares an **Arduino UNO (Microcontroller)** with a **Raspberry Pi (Microprocessor)**.

| **Parameter** | **Microcontroller (Arduino UNO)** | **Microprocessor (Raspberry Pi)** |
|---------------|-----------------------------------|-----------------------------------|
| **CPU Speed** | 16 MHz | 1.5–2.4 GHz (model dependent) |
| **On-chip RAM** | 2 KB SRAM | 1–8 GB LPDDR RAM (external to CPU package) |
| **On-chip Flash** | 32 KB Flash Memory | No on-chip flash; uses microSD card for storage |
| **Operating System** | No operating system (runs a single program) | Linux-based OS (e.g., Raspberry Pi OS) |
| **Typical Use Case** | Sensor interfacing, automation, embedded control | Multimedia, AI, web server, edge computing |
| **Power Consumption** | Low (typically < 0.5 W) | Higher (typically 3–10 W depending on model and load) |
